How to run the program:
1. Create a new database called "Health_and_Fitness_Club_Management_System"
2. Create tables by using "table.sql"
3. Insert the data into the database by using the command inside "SQL command.txt" (It's also  10 commands)
4. Change the information in the "getConnection()"(It's inside the "ClubManagementApp.java")
5. Open the terminal in the dir of this file
6. Compile the file by using "javac ClubManagementApp.java"
7. Run the program by using "java -cp .:postgresql-42.7.0.jar ClubManagementApp.java"
